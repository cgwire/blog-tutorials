## Usage

1. Create a local Kitsu instance with [`kitsu-docker`](https://github.com/cgwire/kitsu-docker)
2. In Kitsu, setup a playlist and add some shots
3. Run the Python script: `python3 script.py`
4. Select a production and a playlist
5. Notice how the shots are stored in a compressed archive for sharing.
